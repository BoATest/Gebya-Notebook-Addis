# Gebya Product Requirements Document (PRD)
## Version: MVP 1.0 - Execution Ready
## Date: 2024

---

## 1. Product Overview

**Name**: Gebya (ገበያ) - "Market"
**Tagline**: "Your business notebook, but smarter"
**Core Promise**: "Know today's money and who owes you"

### Problem
70% of Ethiopian micro-businesses use paper notebooks that:
- Get lost/damaged
- Can't calculate true profit (revenue - cost)
- Lose track of credit (Merro) payments
- Expose sensitive data to customers

### Solution
Digital notebook that:
- Calculates true profit automatically
- Hides money from prying eyes (privacy toggle)
- Tracks credit with partial payments
- Works 100% offline

---

## 2. Target User

**"Ato Kebede"** - Small shop owner
- Phone: Basic Android ($100-200)
- Connection: Intermittent 3G
- Literacy: Grade 8-12 (prefers simple buttons, voice optional)
- Fear: Tax authority, customers seeing profit

---

## 3. Core Principles

1. **30-Second Rule**: Any action completes in 30 seconds
2. **Offline-First**: 100% functional without internet
3. **Privacy-Default**: Money hidden until user reveals it
4. **Notebook Feel**: Not accounting software
5. **Data Moat**: Capture cost prices (optional) for future credit scoring
6. **Ethiopian Context**: Amharic UI, Ethiopian dates, Birr currency

### Tables

**transactions**
- `id`: Auto-increment primary key
- `type`: 'sale' | 'expense' | 'credit_given'
- `amount`: Total selling price
- `item_name`: String
- `cost_price`: Optional, per unit
- `quantity`: Default 1
- `profit`: Calculated: (amount - (cost_price * quantity))
- `is_credit`: Boolean
- `customer_id`: Link to customers (if credit)
- `created_at`: Timestamp
- `ethiopian_date`: String for display

**customers**
- `id`: Primary key
- `name`: String
- `phone`: Optional
- `total_debt`: Cached calculated field

**credit_records**
- `id`: Primary key
- `customer_id`: Foreign key
- `original_amount`: Total debt
- `paid_amount`: Sum of payments
- `remaining_amount`: Calculated
- `due_date`: Timestamp
- `status`: 'active' | 'paid' | 'overdue'

**settings**
- `key`: 'privacy_mode', 'language', etc.
- `value`: Setting value

## 3. Key Functions

### Profit Calculation
```javascript
function calculateProfit(transactions) {
  const sales = transactions.filter(t => t.type === 'sale');
  
  const revenue = sales.reduce((sum, t) => sum + t.amount, 0);
  const costs = sales.reduce((sum, t) => {
    return sum + ((t.cost_price || 0) * (t.quantity || 1));
  }, 0);
  
  return {
    revenue,
    costs,
    profit: revenue - costs,
    hasCostData: sales.some(t => t.cost_price > 0)
  };
}
```

function getCreditStatus(dueDate) {
  const daysUntil = Math.floor((dueDate - Date.now()) / (1000 * 60 * 60 * 24));
  
  if (daysUntil < 0) return { status: 'overdue', color: 'red' };
  if (daysUntil <= 2) return { status: 'due_soon', color: 'yellow' };
  return { status: 'ok', color: 'green' };
}

## 8. Export to CSV

```javascript
function exportToCSV(transactions) {
  const headers = ['Date', 'Type', 'Item', 'Quantity', 'Amount', 'Cost', 'Profit', 'Customer'];
  
  const rows = transactions.map(t => [
    t.ethiopian_date,
    t.type,
    t.item_name,
    t.quantity,
    t.amount,
    t.cost_price || '',
    t.profit || '',
    t.customer_name || ''
  ]);
  
  const csv = [headers, ...rows]
    .map(row => row.join(','))
    .join('\n');
  
  const blob = new Blob([csv], { type: 'text/csv' });
  const url = URL.createObjectURL(blob);
  
  const a = document.createElement('a');
  a.href = url;
  a.download = `gebya-backup-${new Date().toISOString().split('T')[0]}.csv`;
  a.click();
}
```

import { useState, useEffect } from 'react';

function PrivacyToggle({ value, label = "Today's Money", currency = 'birr' }) {
  const [visible, setVisible] = useState(false);

  // Auto-hide after 30 seconds
  useEffect(() => {
    if (visible) {
      const timer = setTimeout(() => setVisible(false), 30000);
      return () => clearTimeout(timer);
    }
  }, [visible]);

  // Hide when app minimizes (visibility change)
  useEffect(() => {
    const handleVisibilityChange = () => {
      if (document.hidden) setVisible(false);
    };
    document.addEventListener('visibilitychange', handleVisibilityChange);
    return () => document.removeEventListener('visibilitychange', handleVisibilityChange);
  }, []);

  const displayValue = visible ? `${value} ${currency}` : '****';

  return (
    <div 
      onClick={() => setVisible(!visible)}
      className="bg-white rounded-2xl p-6 shadow-md cursor-pointer active:scale-95 transition-transform"
    >
      <div className="flex justify-between items-center mb-2">
        <span className="text-gray-500 text-sm uppercase tracking-wide">{label}</span>
        <span className="text-2xl">{visible ? '👁️' : '👁️‍🗨️'}</span>
      </div>
      <div className={`text-4xl font-bold text-center ${visible ? 'text-green-600' : 'text-gray-400'}`}>
        {displayValue}
      </div>
      {!visible && (
        <div className="text-center text-gray-400 text-sm mt-2">Tap to reveal</div>
      )}
    </div>
  );
}

export default PrivacyToggle;
```

  const startListening = () => {
    if (!('webkitSpeechRecognition' in window) && !('SpeechRecognition' in window)) {
      setError('Voice not supported on this device');
      return;
    }

    const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition;
    const recognition = new SpeechRecognition();

    recognition.lang = 'am-ET'; // Amharic
    recognition.interimResults = false;
    recognition.maxAlternatives = 1;
    recognition.continuous = false;

    recognition.onstart = () => {
      setListening(true);
      setError(null);
    };

    recognition.onresult = (event) => {
      const transcript = event.results[0][0].transcript;
      onResult(transcript);
      setListening(false);
    };

    recognition.onerror = (event) => {
      setError('Could not hear. Try again.');
      setListening(false);
    };

    recognition.onend = () => {
      setListening(false);
    };

    recognition.start();
  };

  return (
    <button
      onClick={startListening}
      disabled={listening}
      className={`p-3 rounded-full ${listening ? 'bg-red-500 animate-pulse' : 'bg-blue-500'} text-white`}
      title="Speak instead of type"
    >
      {listening ? '🎤 ...' : '🎤'}
      {error && <span className="text-xs block text-red-600">{error}</span>}
    </button>
  );
}

export default VoiceButton;

import PrivacyToggle from './PrivacyToggle';

function ProfitCard({ transactions }) {
  // Calculate profit
  const sales = transactions.filter(t => t.type === 'sale');
  const revenue = sales.reduce((sum, t) => sum + (t.amount || 0), 0);
  const costs = sales.reduce((sum, t) => {
    return sum + ((t.cost_price || 0) * (t.quantity || 1));
  }, 0);
  const profit = revenue - costs;
  const hasCostData = sales.some(t => t.cost_price > 0);

  return (
    <div className="space-y-3">
      <PrivacyToggle 
        value={hasCostData ? profit : revenue} 
        label={hasCostData ? "Today's Profit" : "Today's Sales"}
      />
      
      {/* Breakdown (always visible, but values respect privacy) */}
      <div className="bg-white rounded-xl p-4 shadow-sm">
        <div className="flex justify-between text-sm">
          <span className="text-gray-600">Sales:</span>
          <span className="font-medium text-green-600">{revenue} birr</span>
        </div>
        {hasCostData && (
          <>
            <div className="flex justify-between text-sm mt-2">
              <span className="text-gray-600">Costs:</span>
              <span className="font-medium text-red-500">{costs} birr</span>
            </div>
            <div className="border-t mt-2 pt-2 flex justify-between">
              <span className="font-medium">Profit:</span>
              <span className={`font-bold ${profit >= 0 ? 'text-green-600' : 'text-red-500'}`}>
                {profit} birr
              </span>
            </div>
          </>
        )}
        {!hasCostData && (
          <p className="text-xs text-gray-400 mt-2">
            Add "what you paid" to see true profit
          </p>
        )}
      </div>
    </div>
  );
}

### src/App.jsx
```javascript
import { useState, useEffect } from 'react';
import Layout from './components/Layout';
import { db } from './db';

function App() {
  const [transactions, setTransactions] = useState([]);
  const [loading, setLoading] = useState(true);

  // Load transactions on mount
  useEffect(() => {
    loadTransactions();
  }, []);

  const loadTransactions = async () => {
    try {
      const all = await db.transactions.toArray();
      // Sort by newest first
      all.sort((a, b) => b.created_at - a.created_at);
      setTransactions(all);
    } catch (error) {
      console.error('Failed to load transactions:', error);
    } finally {
      setLoading(false);
    }
  };

  const addTransaction = async (transaction) => {
    try {
      // Add Ethiopian date
      const now = new Date();
      const ethiopianMonths = ['መስከረም', 'ጥቅምት', 'ኅዳር', 'ታህሳስ', 'ጥር', 'የካቲት', 'መጋቢት', 'ሚያዝያ', 'ግንቦት', 'ሰኔ', 'ሐምሌ', 'ነሐሴ', 'ጳጉሜ'];
      // Simplified Ethiopian date (add 7-8 years approximation)
      const ethYear = now.getFullYear() - 7;
      const ethMonth = ethiopianMonths[(now.getMonth() + 4) % 12];
      const ethDay = now.getDate();
      
      const newTransaction = {
        ...transaction,
        ethiopian_date: `${ethDay} ${ethMonth} ${ethYear}`,
      };

      const id = await db.transactions.add(newTransaction);
      const saved = await db.transactions.get(id);
      
      setTransactions(prev => [saved, ...prev]);
    } catch (error) {
      console.error('Failed to save transaction:', error);
      alert('Could not save. Please try again.');
    }
  };

  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center bg-gray-100">
        <div className="text-center">
          <h1 className="text-2xl font-bold text-primary mb-2">ገበያ</h1>
          <p className="text-gray-500">Loading...</p>
        </div>
      </div>
    );
  }

  return (
    <Layout 
      transactions={transactions} 
      onAddTransaction={addTransaction}
    />
  );
}

export default App;
